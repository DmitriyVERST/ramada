# Ramada Novosibirsk — Апарт-отель

# Live: https://dmitriyverst.github.io/ramada/

## Описание проекта

Этот проект представляет собой сайт апарт-отеля Ramada Novosibirsk, выполненный с использованием современных технологий и библиотек для создания удобного и привлекательного пользовательского интерфейса. Основная цель — предоставить гостям возможность легко ознакомиться с предложениями, номерами, услугами и инфраструктурой отеля.

На сайте реализованы такие функции, как интерактивные слайдеры, плавная прокрутка, динамическая анимация текста и элементов, а также фильтрация и сортировка контента. Это обеспечивает комфортное взаимодействие пользователей с сайтом на любых устройствах и делает навигацию интуитивно понятной.

## Используемые технологии

- **HTML**, **CSS**, **JavaScript** — основы веб-разработки  
- **Swiper** — модуль для реализации слайдеров и каруселей изображений  
- **Lenis** — библиотека для плавной прокрутки страниц  
- **Splitting** — инструмент для анимации текста и элементов  
- **GSAP** — библиотека для создания анимаций и эффектов переходов  
- **Isotope** — плагин для фильтрации и сортировки контента  
- **jQuery** — упрощение работы с DOM и обработкой событий
